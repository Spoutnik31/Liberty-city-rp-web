Les regles à Suivre

1. Respect et Comportement

  1. Respect mutuel : .Chaque membre du serveur doit respecter les autres, sans discrimination liée à l'âge, le sexe, l'origine, la religion ou l'orientation sexuelle.

  2. Aucune insulte : Les propos injurieux, offensants, ou diffamatoires ne seront pas tolérés.

  3. Pas de harcèlement : Le harcèlement, sous quelque forme que ce soit (spam, intimidations, pressions), est interdit.

  4. Langage approprié : Un langage correct est demandé dans les discussions publiques. Les discussions privées ne doivent pas transgresser les règles fondamentales de respect.
